# Small-Shop-Helper
Simple command-line application built in Python for a fictional business to calculate purchase totals and automatically apply discounts. Created as part of the ELATT Software Development programme.

# Overview

Small Shop Helper is a Python console application created as part of the ELATT Software Development programme.

The program helps shop staff calculate a customer's total purchase by reading product information from a CSV file. It automatically applies a discount when the order subtotal reaches a specified amount and generates an invoice for the customer.

This project was created to practise Python programming fundamentals, working with external data files, and building simple business applications.

# Features

- View a list of available products
- Check current inventory levels
- Add items to a shopping cart using Product IDs
- Calculate the total purchase price
- Automatically apply a discount when the subtotal is £30 or more
- Generate an invoice at checkout
- Read product information from a CSV file

# Technologies Used

- Python
- Visual Studio Code
- CSV (Comma Separated Values)

# How to Run

1. Download or clone this repository.
2. Ensure `products.csv` is in the same folder as the Python program.
3. Open the project in Visual Studio Code.
4. Run the Python file.
5. Follow the on-screen menu.

### Menu Options

| Option | Description |
| `p` | View the product list |
| `i` | View available inventory |
| `Enter` | Add an item using its Product ID |
| `q` | Print the invoice and exit |
